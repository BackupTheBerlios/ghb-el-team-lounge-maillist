On 3/28/07, Λούρδας Βασίλειος <lourdas_v at yahoo.gr> wrote:

> Πως προτείνετε να οργανωθούμε εδώ; Με ποια αρχεία θα μπορούσαμε να
> ξεκινήσουμε, το σημαντικό καθορίζεται ίσως από το τι θα ήταν πιο χρήσιμο για
> τον τελικό χρήστη (το handbook, τα howtos, τι...).

Θα προτεινα να ξεκινουσαμε με κειμενα που (1) χρειαζεται κανείς να
διαβασει *πριν* αποφασισει οτι θελει να εγκαταστησει Gentoo (ωστε να
διευκολυνουμε την αποφαση :) ), και που (2) ισως ενδιαφερουν γενικως
και οχι μονο τους χρηστες Gentoo.

Με αυτο το σκεπτικο, θα ελεγα οτι το handbook εχει χαμηλοτερη
προτεραιοτητα γιατι αν κανεις το παρει αποφαση, θα πρεπει ετσι κι
αλλιως να μιλα αγγλικα. Βεβαια, το handbook περιεχει και πραγματα
γενικου ενδιαφεροντος, αλλά ισως λογω του μεγεθους του καλυτερα να το
αφηναμε για λιγο πιο μετα...

Θα προτεινα λοιπον να ξεκινουσαμε με τα FAQ:

http://www.gentoo.org/doc/en/index.xml?catid=faq

Βλεπω (π.χ. στα τσεχικα) οτι δεν ειναι αναγκη να ειναι *ολα* τα FAQ
μεταφρασμενα για να αποκτησει μια γλωσσα παρουσια. Θα μπορουσαμε ετσι
να αποφυγουμε μεταφραση (τουλαχιστον σε πρωτη φαση) των Alpha, Sparc
και MIPS, που μαλλον δεν εχουν μεγαλη παρουσια στην Ελλαδα, ακομη και
του PowerPC που ειναι το πιο μεγαλο.

Μετα (ή ισως και πρωτα;) θα ελεγα να καναμε τα κειμενα στο section
"Other Installation Related Documentation" εδω:

http://www.gentoo.org/doc/en/index.xml?catid=install

που ειναι και γενικοτερου ενδιαφεροντος (οχι μόνο Gentoo). Μετά
πιανόμαστε και με τα HOWTO:

http://www.gentoo.org/doc/en/index.xml?catid=desktop

Νομιζω οτι απο Upgrade Guides και μετα:

# Upgrade Guides
# Gentoo System Documentation
# System Administration Documentation
# Gentoo Development Documentation
# Project Specific Documentation
# Other Documentation

δεν χρειαζονται ελληνικη μεταφραση, τουλαχιστον σε πρωτη φαση.


> Προτείνω επίσης, να έρθουμε σε επαφή με κάποιον υπεύθυνο του Gentoo
> documentation, προκειμένου να μας πει και τη δική του άποψη περί αυτού
> και να συνεχίσουμε κατάλληλα.

Καλη ιδεα, οπως επισης και να δουμε ποια μερη πρωτομεταφραστηκαν σε
αλλες γλωσσες (το εκανα λιγακι πριν πω τα παραπανω).

Νομιζω οτι το να εχουμε τα ελληνικα ως γλωσσα μεταφρασης στις κυριες
σελιδες του Gentoo φαινεται ακομη καλυτερο απο το GWN :)
