   Από αύριο το απόγευμα και για 15 μέρες θα μπορώ να ασχοληθώ πιο ενεργά με 
τις μεταφράσεις λόγω των διακοπών του Πάσχα. Θα κοιτάξω ότι χρειάζεται να 
γίνει πρώτα μετάφραση και θα βάλω μάλλον μέσα στο commit.txt από το svn 
κάποιον αριθμό προτεραιότητας. Το commit.txt μπορεί να χρησιμοποιηθεί σαν το 
history.t2t που έχουμε για το gwn για να μπορούμε να βλέπουμε ποιος 
μεταφράζει τι και να μην γίνεται διπλή δουλειά. Μπορούμε μάλιστα αν κάποιο 
κείμενο είναι μεγάλο να το αναλάβουν 2 ή 3 άτομα (αν και δεν θυμάμαι κάποιο 
τόσο μεγάλο για να δικαιολογήσει τα 3 άτομα). Σκοπός μας θα πρέπει να είναι 
να γίνουν μετάφραση τα πρώτα κεφάλαια που αναφέρονται στην εγκατάσταση του 
βασικού συστήματος. Τα επιπλέον που αφορούν το desktop μετά.
   Για την λήψη των μεταφράσεων και τον έλεγχο κατά πόσο έχει αλλάξει κάποιο 
κείμενο θα μπορούσαμε να σκαρφιστούμε κανένα script όπως αυτά που έφτασαν 
στην gwn-el τελευταία. Προτιμάτε να υπάρχουν όλα τα xml μέσα στο svn ή να 
τραβάμε εμείς όσα γίνονται μετάφραση κάθε φορά και να τα προσθέτουμε μέσα στο 
svn;
   Τα αρχεία του προγράμματος εγκατάστασης θα μπούνε και αυτά στο svn σε έναν 
ξεχωριστό φάκελο για μπορούμε να τα δουλέψουμε και εκείνα. Τελικά μου είχαν 
μείνει αρκετά αμετάφραστα γιατί ενώ έχω ήδη μεταφράσει τα 2 από τα 3 συνολικά 
αρχεία του, αυτό που έχει μείνει έχει συνολικά 350 περίπου μηνύματα και έχω 
μεταφράσει περίπου τα 100. Γνωρίζουμε περίπου ημερομηνία κυκλοφορίας νέας 
έκδοσης gentoo να βάλουμε κάποιο πρόγραμμα; Καλό θα ήταν αν μπορούσαμε να 
έχουμε παρουσία στην επόμενη έκδοση. Έστω και κουτσή :-)
   Αυτά τα ολίγα από εμένα. Περιμένω τις απόψεις σας.
